Jupyter Lab

Version 3.0.4를 인스톨 하려면

conda install -c conda-forge jupyterlab=3

라즈베리에서 주피터랩을 사용하기 위해서는 주피터를 인스톨한 후에 아래와 같이 추가해야 한다.

sudo -H pip3 install jupyterlab

sudo jupyter serverextension enable –py jupyterlab –system

  • c.Spawner.default_url = ‘/user/:username/lab’

  • 왼쪽 Table of Content를 보면 목차를 볼수 있다.

주피터 랩 서버로 돌리기

jupyter lab –ip=“0.0.0.0” –no-browser –port=8889
